## 0.1.0

- Initial release
- `LineupBuilder.display` — Read-only two-team match lineup
- `LineupBuilder.single` — Read-only single-team full-pitch view
- `LineupBuilder.editable` — Drag-and-drop formation builder (single team)
- `LineupBuilder.tactical` — Two-team tactical board for analysis
- `PlayerPosition` enum (goalkeeper, defender, midfielder, forward)
- Animated position transitions on formation change
- Drag trail (meteor effect) from start to current position
- Configurable haptic feedback (`HapticConfig`)
- Custom `playerNodeBuilder` support
